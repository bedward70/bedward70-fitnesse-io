This is helpful library of Fitnesse fixtures for working with files.

IO section:
1. B70DoFixture contains saveSymbol and readSymbol methods;
2. B70FileFixture contains readFromFile, readByteFromFile and saveToFile methods.
3. B70JoinFixture contains string join methods
4. B70PropertyFixture contains get property method.

Row section:
1. B70PropertyOrderedRowFixture views properties as a table and can read and check properties.
2. B70SectionPropertyOrderedRowFixture views windows section style properties as a table and can read and check properties.
